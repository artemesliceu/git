# git

Resum de commandes esencials de git.

```
# Inicialitzar un repositori local
git init
```

```
# Connectar amb el repositori remot (de GitHub)
git remote add origin git@github.com:nomusuari/nomrepositori.git
git branch -M main
git push -u origin main
```

```
# Afeguir tots els fitxers
git add .
```

```
# Afeguir un fitxer
git add README.md
```

```
# Fer un commit (canvi)
git commit
```

```
Fer un commit (amb el missatge de commit)
git commit -m "Descripció del commit"
```

```
# Pujar un canvia al repositori remot (a GitHub)
git push
```

```
# Baixar els canvis del repositori remot (a GitHub) al repositori local (a la teva màquina)
git pull
```
